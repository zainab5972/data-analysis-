# Traffic Collisions Data Analysis

This project contains analysis of traffic collision data from Toronto's open data portal.

## Dataset Overview

**File:** `Traffic_Collisions_Open_Data.csv`
- **Size:** 790,727 records
- **Time Period:** 2014 - Present
- **Source:** Toronto Police Service Open Data Portal
- **Content:** Traffic collision reports with location, timing, and severity information

## Data Schema

| Column | Description | Type |
|--------|-------------|------|
| OBJECTID | Unique identifier | Integer |
| EVENT_UNIQUE_ID | Event identification code | String |
| OCC_DATE | Date and time of collision | DateTime |
| OCC_MONTH | Month of occurrence | String |
| OCC_DOW | Day of week | String |
| OCC_YEAR | Year of occurrence | Integer |
| OCC_HOUR | Hour of occurrence (0-23) | Integer |
| DIVISION | Police division code | String |
| FATALITIES | Number of fatalities | Integer |
| INJURY_COLLISIONS | Injury collision flag (YES/NO) | String |
| FTR_COLLISIONS | Failure to Remain collision flag (YES/NO) | String |
| PD_COLLISIONS | Property Damage collision flag (YES/NO) | String |
| HOOD_158 | Neighborhood ID | String |
| NEIGHBOURHOOD_158 | Neighborhood name | String |
| LONG_WGS84 | Longitude coordinate | Float |
| LAT_WGS84 | Latitude coordinate | Float |
| AUTOMOBILE | Automobile involved (YES/NO/N/R) | String |
| MOTORCYCLE | Motorcycle involved (YES/NO/N/R) | String |
| PASSENGER | Passenger vehicle involved (YES/NO/N/R) | String |
| BICYCLE | Bicycle involved (YES/NO/N/R) | String |
| PEDESTRIAN | Pedestrian involved (YES/NO/N/R) | String |
| x | X coordinate (projected) | Float |
| y | Y coordinate (projected) | Float |

## Files in this Project

### Analysis Scripts
- `data_distribution_analysis.py` - Comprehensive data distribution analysis with visualizations

### Data Files
- `Traffic_Collisions_Open_Data.csv` - Main dataset containing all collision records

### Output Files
- `traffic_collisions_distribution.png` - Visualization of data distributions (generated by analysis script)

## Key Features of the Analysis

The `data_distribution_analysis.py` script provides:

1. **Temporal Analysis**
   - Yearly collision trends
   - Monthly patterns
   - Day-of-week distribution
   - Hourly patterns

2. **Geographic Analysis**
   - Police division breakdown
   - Neighborhood hotspots
   - Coordinate validation

3. **Collision Severity**
   - Fatalities distribution
   - Injury vs. property damage
   - Hit-and-run incidents

4. **Vehicle Type Analysis**
   - Automobiles, motorcycles, bicycles
   - Pedestrian involvement
   - Multi-vehicle collisions

## Getting Started

### Prerequisites
```bash
pip install pandas numpy matplotlib seaborn
```

### Running the Analysis
```bash
python data_distribution_analysis.py
```

This will:
- Generate comprehensive distribution statistics
- Create visualization plots
- Save results as `traffic_collisions_distribution.png`

## Data Quality Notes

- Some records have missing coordinates (marked as 0,0)
- Vehicle involvement may be marked as "N/R" (Not Reported)
- NSA (Not Specified Area) appears in some location fields
- Data quality varies by year and reporting standards
